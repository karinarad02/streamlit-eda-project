# Streamlit EDA App

Aplicație Streamlit pentru analiză exploratorie a datelor (EDA).

## Funcționalități:

- Încărcare CSV / Excel
- Filtrare numerică și categorică
- Statistici descriptive
- Analiza valorilor lipsă
- Histogramă & boxplot
- Analiză categorică
- Matrice corelații, scatter plot
- Detectare outlieri (IQR)

## Cum se rulează local:

pip install -r requirements.txt
streamlit run app.py
